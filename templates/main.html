Api para extraer texto de páginas de noticias en español.
 
 

Herramienta de scraping para extraer el texto principal de paginas web dedicadas a noticias. Funciona como api mediante el método POST llamando a la url: http://extracttextpython.appspot.com/api/. No necesita token o loguearse.
Esta versión aún está en desarrollo.

 
 

Pruebe usted mismo:

{{data}}

 
 

Ejemplo de llamada a la Api en Python:

 

import requests

data = {"url":"https://www.subrayado.com.uy/uruguay-le-gano-2-1-hungria-el-amistoso-que-se-jugo-budapest-n571861"}

url = "http://extracttextpython.appspot.com/api/"

r = requests.post(url, data=data, allow_redirects=True)

print(r.content)
